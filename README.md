# 🎂 Age Calculator

A simple and responsive **Age Calculator** web application built using **HTML, CSS, and JavaScript**. It allows users to enter their date of birth and instantly calculates their exact age in **years, months, and days**.

---

## 📌 Features

- Enter Date of Birth using a date picker
- Calculates age in years, months, and days
- Validates user input
- Prevents selecting future dates
- Simple and responsive user interface
- Fast and lightweight application

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- JavaScript Date Object
- DOM Manipulation
